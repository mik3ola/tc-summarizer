# Extension Icons

This directory should contain the following icon files (PNG format):

- **icon16.png** - 16x16 pixels (required)
- **icon48.png** - 48x48 pixels (required)
- **icon128.png** - 128x128 pixels (required)

## Icon Design Guidelines

- Use a simple, recognizable design that represents legal documents/summaries
- Ensure icons are readable at small sizes (16x16)
- Use transparent background
- Consider using a document icon with a checkmark or summary symbol
- Maintain consistent styling across all sizes

## Tools for Creating Icons

- **Figma** - Free design tool
- **Canva** - Easy icon creation
- **Icon generators** - Search for "Chrome extension icon generator"
- **Hire a designer** - Fiverr, Upwork, etc.

## Quick Icon Ideas

- Document with a magnifying glass
- Document with a checkmark
- Legal scroll with summary lines
- Terms document with "T&C" text
- Document with a sparkle/summary symbol

## Testing

After adding icons, test the extension:
1. Load the extension in Chrome
2. Check that icons appear correctly in:
   - Extension toolbar
   - Extensions page (chrome://extensions)
   - Chrome Web Store listing (after upload)
